# Web Specs

The base URI is [http://localhost:8080](http://localhost:8080).

## Functionalities

- Home page
  - An empty home page
- Owner
  - All
    - List all owners
    - Search an owner
    - View Owner details
    - Edit Owner
    - Pets and Visits
      - View Pet
      - Edit Pet
      - Add Visit
      - View preview visits
  - Register
    -  Add a new Owner
- Veterinarians
  - View Veterinarians
